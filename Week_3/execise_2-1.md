## IF

#### Play with if statement

An if ... elif ... elif ... else sequence is a substitute for the switch or case or default statement found in other language.

    x = 10
    if x < 0:
        print("negative value")
    elif x == 0:
        print("zero")
    elif x == 1:
        print("single")
    else:
        print("more")
